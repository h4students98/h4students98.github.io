Step 3: JavaScript to Load JSON and Render Content

In your Dropdownfunctions.js or a new JavaScript file, you'll include the function to load the JSON and render the content:

javascript

function loadSubject(subject) {
  const language = 'en'; // This could be dynamic based on user selection
  const filePath = `../data/${subject}_${language}.json`;

  fetch(filePath)
    .then(response => response.json())
    .then(data => {
      const contentArea = document.getElementById('contentArea');
      contentArea.innerHTML = ''; // Clear existing content
      
      data.forEach(section => {
        const sectionElement = document.createElement('div');
        sectionElement.innerHTML = `<h3>${section.year}</h3>`;
        const rowElement = document.createElement('div');
        rowElement.className = 'row';
        
        section.documents.forEach(doc => {
          const docElement = document.createElement('div');
          docElement.className = 'fourcol';
          docElement.innerHTML = `<a href="${doc.url}">${doc.name}</a>`;
          rowElement.appendChild(docElement);
        });
        
        sectionElement.appendChild(rowElement);
        contentArea.appendChild(sectionElement);
      });
    })
    .catch(error => console.error('Error loading the document list:', error));
}

This JavaScript:

    Loads the correct JSON file based on the subject clicked.
    Parses the JSON to dynamically create the HTML structure with the document links.
    Injects this content into #contentArea.

Step 4: Adapt for Multiple Languages

To adapt this for multiple languages, you could modify the language variable in the loadSubject function based on user selection. This might involve setting a global variable when the user selects a language, storing the choice in localStorage, or appending it to the URL as a query parameter and reading it in your script.

This approach greatly simplifies your HTML and centralizes content management within JSON files, making updates and language management much more straightforward.
